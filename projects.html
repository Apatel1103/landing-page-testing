Pathfinding for a Robotic Robot

Developed an algorithm to use on a Boston Dynamics Spot robot. This algorithm is used to ensure the robot can guide a human approximately one meter behind the robot and navigates the person through various environments including obstacles both stationary and dynamic.

Excercise Tracker

Brief project to track the progress of certain exercises by tracking the time of doing a certain movement.
Utilizes Mongoose library and MongoDB to create a project that is very easy to use for a user.
